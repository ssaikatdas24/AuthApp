# AuthApp

A Node.js-based authentication backend using MongoDB and Express.

## Features
- User Registration/Login
- JWT Authentication
- Role-based access
- MongoDB with Mongoose

## Technologies Used
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt
- dotenv

## Getting Started

### Prerequisites
- Node.js
- MongoDB

### Installation
```bash
git clone https://github.com/ssaikatdas24/AuthApp.git
cd AuthApp
npm install
